# TOEIC 600 Flashcard One File

Bản này chỉ có 1 file index.html.

Cách dùng:
1. Giải nén ZIP.
2. Mở VS Code.
3. File > Open Folder.
4. Chọn thư mục toeic_flashcard_onefile.
5. Click vào index.html.
6. Click phải index.html > Open with Live Server.

Vì toàn bộ CSS, JavaScript và 600 từ đã được nhúng trong index.html nên không cần style.css, app.js, vocab.json riêng.
